# 国际化 (i18n) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 实现一个轻量级的基于字典的国际化方案，支持英文，且保持原有 `'.tr'` 语法不变。

**Architecture:** 提取翻译字段到 `translations.dart`，通过 `AppLanguage` 全局静态类管理语言类型，通过 `PreferencesProvider` 刷新 UI，通过 `StorageService` 持久化。

**Tech Stack:** Dart, Flutter, Hive (Storage)

---

### Task 4: 启动时初始化全局语言并改造 `l10n.dart`

**Files:**
- Modify: `lib/main.dart`
- Modify: `lib/config/l10n.dart`

- [ ] **Step 1: 在 `main.dart` 初始化 `AppLanguage`**

在 `lib/main.dart` 的 `main()` 函数中，`await storageService.initialize();` 后面添加：
```dart
    // 初始化语言
    AppLanguage.currentLanguage = storageService.language;
```
并且在文件顶部加上 `import 'package:ultimate_wheel/config/translations.dart';`。

- [ ] **Step 2: 在 `PreferencesProvider` 更新时同步更新 `AppLanguage`**

修改 `lib/providers/preferences_provider.dart` 中的 `setLanguage` 方法，加上更新 `AppLanguage` 的逻辑：
```dart
  /// 设置语言
  Future<void> setLanguage(String lang) async {
    await _storageService.setLanguage(lang);
    _language = lang;
    import 'package:ultimate_wheel/config/translations.dart'; // 确保顶部有引入
    AppLanguage.currentLanguage = lang;
    notifyListeners();
  }
```
*确保文件顶部有 `import 'package:ultimate_wheel/config/translations.dart';`*

- [ ] **Step 3: 修改 `l10n.dart` 拦截实现**

重写 `lib/config/l10n.dart`：
```dart
import 'package:ultimate_wheel/config/translations.dart';

extension StringL10n on String {
  String get tr {
    if (AppLanguage.currentLanguage == 'en') {
      return appTranslations['en']?[this] ?? this;
    }
    return this;
  }
}
```

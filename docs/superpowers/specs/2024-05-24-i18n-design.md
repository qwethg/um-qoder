# 飞盘之轮 (Ultimate Wheel) 国际化 (i18n) 设计方案

## 1. 目标
为项目增加英语版本支持，同时保持现有代码架构不变，特别是 UI 层的原有 `'.tr'` 语法保持不变。采用轻量级字典拦截方案，确保 AI 实施的安全性和代码稳定性。

## 2. 核心架构
- **静态全局语言状态**：建立一个轻量的全局变量记录当前语言（避免 `l10n.dart` 扩展对 Context 产生依赖）。
- **底层字典拦截**：扩展现有的 `StringL10n`，根据全局的当前语言状态去字典里匹配翻译。
- **配置字典**：建立静态的 `appTranslations` 字典，配置所有使用到的翻译词条。
- **全局状态管理与持久化**：在现有的 `PreferencesProvider` 和 `StorageService` 中增加对 `language` 状态的支持，以驱动界面刷新和保存用户设置。

## 3. 具体实现步骤
1. 新建 `lib/config/translations.dart`：
   - 定义静态类 `AppLanguage` 保存 `static String currentLanguage = 'zh';`。
   - 定义字典 `const Map<String, Map<String, String>> appTranslations = {'en': {...}};`。
2. 升级 `lib/services/storage_service.dart`：增加读取/保存 `language` 的逻辑。
3. 升级 `lib/providers/preferences_provider.dart`：
   - 增加 `String _language` 状态。
   - 增加 `setLanguage(String lang)` 方法：保存到 storage、更新 `AppLanguage.currentLanguage`、触发 `notifyListeners()`。
4. 升级 `lib/main.dart`：在应用启动时（`StorageService` 初始化后），同步将读取到的语言写入 `AppLanguage.currentLanguage`。
5. 升级 `lib/config/l10n.dart`：修改 `.tr` 的实现，从 `AppLanguage.currentLanguage` 拿到语种并在字典中做查找。
6. 升级 `lib/screens/settings/settings_screen.dart`：在设置页面增加一个“语言切换”列表项（Dropdown / SegmentedButton）。

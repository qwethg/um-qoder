## 现状与问题
- 固定三列与固定宽高比：`lib/screens/settings/radar_theme_manager_screen.dart:88-93, 110-115` 使用 `SliverGridDelegateWithFixedCrossAxisCount(crossAxisCount: 3, childAspectRatio: 0.85)`，导致不同窗口宽度下卡片尺寸僵硬，留白明显，且无法自适应。
- 缩略图固定像素：`lib/widgets/radar_theme_preview.dart:17-21, 46-59` 预览使用固定 `size = 120`，不根据父约束缩放，造成卡片内容与网格格子的尺寸不匹配。
- 选中态边框包裹整个格子：`lib/widgets/radar_theme_preview.dart:33-42` 的 `Container` 在网格中会占满父约束，使得描边显得过大且不协调。

## 目标
- 自适应列数与卡片尺寸，确保在窄屏/宽屏都美观。
- 预览雷达图根据格子宽度自动缩放，保持一致的视觉比例。
- 优化选中态样式与间距，统一卡片高度、对齐与留白。

## 改动范围
- `lib/screens/settings/radar_theme_manager_screen.dart`：两处 GridView 的委托与尺寸逻辑。
- `lib/widgets/radar_theme_preview.dart`：移除绝对像素依赖，基于父约束计算预览大小，并优化选中态样式。
- `lib/widgets/ultimate_wheel_radar_chart.dart`：无需改绘制逻辑；通过外层 `FittedBox`/约束驱动实现缩放。

## 实施方案
### 1) 响应式网格委托
- 用 `LayoutBuilder` 根据容器宽度动态计算列数或最大单元宽度；改为 `SliverGridDelegateWithMaxCrossAxisExtent`，保证不同屏宽下单元数量和尺寸自适应。
- 同时调优间距；移除固定 `childAspectRatio` 或设置更合理的比值（考虑标题两行文本高度）。

```dart
// lib/screens/settings/radar_theme_manager_screen.dart
Widget _buildThemeGrid(...) {
  return LayoutBuilder(
    builder: (context, constraints) {
      final w = constraints.maxWidth;
      final maxExtent = w < 600 ? 200 : w < 900 ? 220 : w < 1200 ? 240 : 260;
      return GridView.builder(
        shrinkWrap: true,
        physics: const NeverScrollableScrollPhysics(),
        gridDelegate: SliverGridDelegateWithMaxCrossAxisExtent(
          maxCrossAxisExtent: maxExtent,
          crossAxisSpacing: 16,
          mainAxisSpacing: 16,
          childAspectRatio: 0.95,
        ),
        itemCount: themes.length,
        itemBuilder: (context, index) {
          final theme = themes[index];
          return RadarThemePreview(
            theme: theme,
            isSelected: theme.id == currentTheme.id,
            onTap: () => onSelect(theme),
          );
        },
      );
    },
  );
}
```
- 自定义主题网格同样替换为上述逻辑（含删除按钮）。

### 2) 预览缩略图自适应
- `RadarThemePreview` 使用 `LayoutBuilder` 读取父约束，计算预览雷达图的边长；用 `FittedBox` 包装 `UltimateWheelRadarChart`，保持优雅缩放并避免溢出。
- 统一卡片高度：通过内层 `AspectRatio` 或固定内边距+文本高度预留，使网格项高度稳定。

```dart
// lib/widgets/radar_theme_preview.dart
class RadarThemePreview extends StatelessWidget {
  final RadarTheme theme;
  final bool showName;
  final VoidCallback? onTap;
  final bool isSelected;

  const RadarThemePreview({
    super.key,
    required this.theme,
    this.showName = true,
    this.onTap,
    this.isSelected = false,
  });

  @override
  Widget build(BuildContext context) {
    final sampleScores = <String, double>{
      for (var ability in AbilityConstants.abilities)
        ability.id: 6.5 + (ability.order % 3) * 1.0,
    };

    return LayoutBuilder(
      builder: (context, constraints) {
        final tileWidth = constraints.maxWidth;
        final previewSize = tileWidth * 0.8; // 为标题预留高度
        return InkWell(
          onTap: onTap,
          borderRadius: BorderRadius.circular(12),
          child: Padding(
            padding: const EdgeInsets.all(8),
            child: Column(
              mainAxisSize: MainAxisSize.min,
              children: [
                // 预览图区域（保持正方形并自适应）
                SizedBox(
                  width: previewSize,
                  height: previewSize,
                  child: FittedBox(
                    fit: BoxFit.contain,
                    child: SizedBox.square(
                      dimension: 240, // 基准尺寸，FittedBox 会缩放
                      child: UltimateWheelRadarChart(
                        scores: sampleScores,
                        size: 240,
                        showLabels: false,
                        showGrid: true,
                        gridLevels: 5,
                        radarTheme: theme,
                        showStroke: isSelected,
                      ),
                    ),
                  ),
                ),
                if (showName)
                  Padding(
                    padding: const EdgeInsets.only(top: 8),
                    child: Text(
                      theme.name,
                      maxLines: 2,
                      overflow: TextOverflow.ellipsis,
                      textAlign: TextAlign.center,
                      style: Theme.of(context).textTheme.labelMedium?.copyWith(
                        fontWeight: isSelected ? FontWeight.w600 : FontWeight.w400,
                      ),
                    ),
                  ),
              ],
            ),
          ),
        );
      },
    );
  }
}
```
- 说明：移除固定 `size` 参数，改为基于可用宽度计算；在不同网格列数下可自然缩放。

### 3) 选中态样式优化
- 用 `InkWell` + `borderRadius` + 轻微阴影或描边只环绕内容，而非占满整个格子。
- 可用 `AnimatedContainer` 增加柔和过渡：选中时描边与阴影平滑出现。

### 4) 文案与间距统一
- 网格间距：`crossAxisSpacing/mainAxisSpacing` 统一为 `16`。
- 文本样式：使用 `labelMedium`，选中加粗但不改变字号，避免跳动。
- 两行省略并居中，保证卡片高度一致。
